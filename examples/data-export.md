# Data Export Options Example

## Session End Menu

After capturing session data, the system offers flexible export:

```
💾 SESSION CAPTURED!

📊 Data Summary:
✅ 12 preferences learned
✅ 3 architectural decisions
✅ 8 files modified
✅ 1 project in progress (67%)
✅ 100% context preserved

⏱️  Duration: 6 hours
🎯 Status: Ready for /new or continue
```

## Export Options

### Option A: Add More Data

```
📝 Great! What else do you have?

Sometimes you have info on:
• Your phone (photos, notes)
• Paper (handwritten notes)
• Your head (ideas not typed)
• Other apps (emails, chats)

Tell me everything to add:
[Input field]
```

**What's included:**
- 📧 Session summary (Markdown)
- 📋 Pending tasks list
- 💾 Modified files (ZIP)
- 🔗 Links to resources


## Real-World Scenario

**User choice breakdown:**

| Choice | When to Use | Result |
|--------|-------------|--------|
| **A** (Add more) | Have notes elsewhere | Richer context |
| **1** (Email) | Want summary in inbox | Email in 30s |
| **2** (Drive) | Need offline backup | Full backup |
| **3** (PDF) | Professional delivery | Formal report |
| **4** (Auto) | Just want to continue | Seamless next time |

## Automatic vs Manual

**Automatic (default):**
- ✅ Zero user action
- ✅ Background sync every 5 min
- ✅ Auto-detect on new session
- ✅ Restore with one confirmation

**Manual (export options):**
- 📧 Share with team
- 💾 Archive completed projects
- 📝 Professional documentation
- 📱 Access on other devices
